#WSSkinStatesTester is a handy tool to test visually all states of any skin classes in Flex 4 (Gumbo).

##Usage
Just follow the instruction posted on WS-Blog: [Testing states of skin classes in Flex 4 using WSSkinStatesTester](http://www.websector.de/blog/2009/09/06/testing-states-of-skin-classes-in-flex-4-using-wsskinstatestester/)
